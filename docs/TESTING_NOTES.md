Tabavoco is a simple taskbar volume control overlay. No account, login, or internet connection is required.

To test:
1. Launch the app - a small volume slider appears anchored to the taskbar.
2. Drag the slider to change system volume. Click the speaker icon to mute/unmute.
3. Right-click the overlay for a context menu with exit and startup options.
4. Drag the overlay to reposition it along the taskbar.
5. If media is playing, play/pause/next/previous buttons appear.

The app runs entirely offline with no backend services.

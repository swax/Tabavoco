# Privacy Policy

**Tabavoco** is a local desktop application that provides a volume control overlay for the Windows taskbar.

## Data Collection

Tabavoco does **not**:

- Collect, store, or transmit any personal information
- Connect to the internet or make any network requests
- Use analytics, telemetry, or tracking of any kind
- Access any data beyond what is required for volume control and window positioning

## Local Storage

Tabavoco stores a single configuration file (`appsettings.json`) on your device containing your window position and app preferences. This file never leaves your device.

## Third-Party Services

Tabavoco does not integrate with or send data to any third-party services.

## Contact

If you have questions about this privacy policy, you can open an issue on the [GitHub repository](https://github.com/TransparentSource/tabavoco).

## Changes

If this policy changes, the updated version will be posted in the repository.
